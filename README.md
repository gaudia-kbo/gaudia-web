# GAUDIA ⚾
> 경기마다 쌓이는 즐거운 순간들 — KBO 집단지성 예측 플랫폼

## 서비스 소개
GAUDIA는 KBO 프로야구 팬들이 경기 승패·선취점·선수 스탯 등을 예측하고,
예측 정확도를 겨루는 팬 참여형 정보 서비스입니다.

## 주요 기능
- ⚾ KBO 실시간 경기 예측 참여
- 📊 예측 정확도 기반 리더보드
- 💬 팀별 커뮤니티 게시판
- 🏆 주간·월간 예측왕 선발

## 기술 스택
- Frontend: HTML / CSS / JavaScript
- Backend: Next.js + Supabase (예정)
- Deploy: Vercel
